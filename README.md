# Company_assets

## Описание предметной области

Данный проект посвящен учету материальных и нематериальных активов предприятия.  
Система позволяет вести полную информацию об активах, их категориях, местах расположения, ответственных сотрудниках и сопутствующих документах.  

Основные возможности системы:  
- Учет оборудования, мебели, техники, программного обеспечения и лицензий.  
- Разграничение прав доступа пользователей по ролям (Пользователь, Менеджер, Администратор).  
- Инвентаризация и маркировка активов.  
- Учет амортизации и переоценки активов.  
- Планирование технического обслуживания.  
- Контроль перемещений между подразделениями.  
- Формирование балансовой отчетности.

Ключевые таблицы базы данных:  
- `assets` — информация о всех активах предприятия.  
- `categories` — справочник категорий активов (ОС, НМА, МПЗ).  
- `locations` — места расположения активов (офисы, склады, цеха).  
- `responsibles` — сотрудники, ответственные за активы.  
- `documents` — документы, связанные с активами (акты приема-передачи, гарантийные талоны).  
- `asset_movements` — история перемещений активов между локациями.
