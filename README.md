# Asset Management System

## Описание проекта

Данное приложение разработано в рамках практической работы.
Система предназначена для управления активами компании.

Приложение реализует клиентскую часть информационной системы и включает:

- окно авторизации пользователей
- окно регистрации
- фирменный стиль интерфейса
- централизованное управление стилями через App.xaml

## Предметная область

Управление активами компании:

- учет оборудования
- учет мебели
- учет программного обеспечения
- учет лицензий
- контроль перемещения активов

## Используемые технологии

- C#
- WPF
- Visual Studio
